# git-branch-manager

## Install

### Build from source

```sh
$ git clone https://github.com/demonkingswarn/git-branch-manager
$ cd git-branch-manager
$ sudo make install
```

### AUR

```sh
$ paru -S git-branch-manager
```

## Usage

```sh
$ git branch-manager
```

## Dependency

- [gum](https://github.com/charmbracelet/gum) - To make selections prettier
